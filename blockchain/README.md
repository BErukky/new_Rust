# CLI Toy Blockchain 🦀⛓️

A simple toy blockchain built from scratch using Rust, running entirely in the terminal.

Built to understand how blockchains actually work under the hood — no libraries doing the heavy lifting, just pure Rust.

## What it does
- Creates and chains blocks together
- Hashes each block using cryptographic concepts
- Validates the integrity of the chain
- Runs entirely in the CLI

## Built with
- Rust
- Cargo

## How to run
```bash
git clone https://github.com/BErukky/your-repo-name
cd your-repo-name
cargo run
```

## What I learned
- Rust structs and enums in real scenarios
- Hashing and data integrity concepts
- Ownership and borrowing applied practically
- How blockchain works from the ground up
